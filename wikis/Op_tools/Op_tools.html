Op_tools is the first mod that Game Mod Studios made

Op_tools is a mod that:

Gives you op tools and armor that almost make you a god by:

  • Adding a new ore, blocks, armor, and tools
  • FabricApi required for the fabric version
  • Op_ore can be found below 0 that you smelt into op_ingot that can be crafted into, for example, op_sword, op_pickaxe, and op_armor (recipes included in the guide book)

Versions:

  • 1.20.1
